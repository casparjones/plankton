# Task-026: Agenten-Workflow Tools

**Epic:** epic-008
**Status:** open
**Rolle:** Developer

## Beschreibung
Neue MCP-Tools für den Agenten-Workflow: Epics verwalten, Tasks zuweisen, Reviews durchführen.

## Akzeptanzkriterien
- [ ] list_epics: Spalten als "Epics" auflisten mit Task-Counts
- [ ] create_task: Task erstellen mit automatischem creator aus Token
- [ ] assign_task: Task einem Worker zuweisen
- [ ] submit_for_review: Task als "ready for review" markieren (Label setzen)
- [ ] get_review_queue: Tasks mit "review" Label auflisten
- [ ] add_comment: Kommentar zu einem Task hinzufügen
- [ ] approve_task: Task in Done-Spalte verschieben
- [ ] reject_task: Task zurück in vorherige Spalte + Kommentar
